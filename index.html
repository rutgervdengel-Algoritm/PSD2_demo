🏗️ Architectuur
Van PSD2 API tot data product — de complete flow
┌─────────────┐ ┌──────────────┐ ┌──────────────────┐ ┌────────────────┐ │ Klant │────▶│ Tink Link │────▶│ Tink API │────▶│ Python Script │ │ (consent) │ │ (OAuth 2.0) │ │ (PSD2 Gateway) │ │ (classificatie)│ └─────────────┘ └──────────────┘ └──────────────────┘ └───────┬────────┘ │ ┌──────────────────────────────────────────────────────┘ ▼ ┌───────────────────┐ │ Data Product │ │ (JSON API) │ └─────────┬─────────┘ │ ┌─────────────┼─────────────┬──────────────┬──────────────┐ ▼ ▼ ▼ ▼ ▼ ┌────────┐ ┌─────────┐ ┌──────────┐ ┌───────────┐ ┌───────────┐ │ Mijn- │ │ Alerts │ │Financial │ │ Risk & │ │ BI / │ │omgeving│ │ & Mail │ │Planning │ │Acceptatie │ │ Analytics │ └────────┘ └─────────┘ └──────────┘ └───────────┘ └───────────┘ Tink API Endpoints (bewezen in sandbox): POST /api/v1/oauth/token → Client credentials grant POST /api/v1/user/create → Sandbox user aanmaken POST /api/v1/oauth/authorization-grant → Authorization code GET /api/v1/providers → Beschikbare banken (NL: 1 test, prod: 20+) POST /api/v1/credentials → Bankconnectie starten GET /api/v1/accounts/list → Accounts ophalen GET /api/v1/search → Transacties ophalen Classificatie Engine: 1. Keyword matching op description (14 categorieën, 80+ keywords) 2. Patroon detectie: interval-analyse met standaarddeviatie 3. Frequentie: maandelijks / kwartaal / halfjaarlijks / jaarlijks 4. Confidence scoring (0.3 - 0.95) 5. Prijswijzigingsdetectie (min/max/richting) 6. Volgende betaling schatting Productie-pad: 1. Tink enterprise account aanvragen (sandbox → productie) 2. Echte NL banken beschikbaar: ING, Rabobank, ABN AMRO, etc. 3. Script draait identiek, alleen credentials wijzigen 4. Classificatie-engine direct inzetbaar op echte transactiedata
